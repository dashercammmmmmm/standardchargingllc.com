How it works

Standard Charging starts with fit, documentation, and responsible routing before any public claims or installation steps.

1. Share the opportunity

Hosts and sponsors submit a short form with location, goals, and context. Drivers can also report charging issues from the support page.

2. Review fit

We look at community value, site visibility, likely driver use, sponsor alignment, and what questions need to be answered before moving forward.

3. Route technical work responsibly

Electrical assessment, permitting, repairs, and installation are handled by licensed installation partners or the responsible equipment/network provider.

4. Launch with proof

When a project is ready, claims, recognition, reporting, and impact recaps should be based on confirmed details and approved project information.

Not sure where to start? Hosts can use the Become a Host form. Sponsors can use the Become a Sponsor form. Drivers can use the Report a Charging Issue form.
